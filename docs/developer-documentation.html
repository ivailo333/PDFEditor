PDFEditor
Документация за програмиста

1. Обща архитектура

PDFEditor е Qt Widgets приложение на C++17 за създаване, четене, визуално редактиране и запис на PDF документи. Проектът е структуриран като qmake проект за Qt Creator чрез файла PDFEditor.pro.

Приложението е направено с условни backend-и, за да може проектът да се отваря и компилира дори когато някои външни зависимости липсват. Ако Qt PDF е наличен, се използва реалният viewer. Ако QPDF е настроен, редакции върху съществуващи PDF файлове се записват като PDF annotation обекти.

3. Зависимости

3.1 Задължителни

  • Qt 6
  • Qt Widgets
  • C++17 compiler

3.2 За преглед и създаване на PDF

  • Qt PDF module

3.3 За редактиране на съществуващ PDF

  • QPDF development package
  • Environment variable QPDF_ROOT, сочещ към папка с include/qpdf/QPDF.hh и lib/

4. Build с Qt Creator

  1. Отворете PDFEditor.pro.
  2. Изберете Qt 6 Desktop Kit.
  3. Ако искате QPDF функционалност, задайте QPDF_ROOT в Projects -> Build Environment.
  4. Изпълнете Build -> Run qmake.
  5. Изпълнете Build -> Rebuild Project.
QPDF_ROOT=C:/vcpkg/installed/x64-mingw-dynamic

5. Условни модули

В PDFEditor.pro се използва qtHaveModule(pdf). Ако Qt PDF е наличен, се компилира PdfDocumentViewQt.cpp. Ако не е наличен, се компилира PdfDocumentViewStub.cpp.

За QPDF се проверява дали съществува $$QPDF_ROOT/include/qpdf/QPDF.hh. Ако файлът е наличен, се компилира QpdfBackend.cpp; иначе се използва QpdfBackendStub.cpp.

6. Текстов редактор

Текстът в PDF страницата се моделира чрез структурата TextEdit. Тя пази:

  • page - индекс на страницата;
  • pageRect - позиция и размер в координатите на PDF страницата;
  • pageSizePoints - размер на страницата в PDF points;
  • text - съдържание;
  • font - шрифт, големина, bold, italic, underline;
  • color - цвят, по подразбиране черен.

При активен режим Edit клик върху страницата създава текстова кутия. За активното поле се показва child QTextEdit, което позволява директно въвеждане и редактиране на текста.

7. Запис на PDF

7.1 Нов PDF

Новосъздаден документ се записва чрез QPdfWriter. Всички текстови кутии се рисуват върху A4 страници с QPainter::drawText, като се запазват шрифт, големина и стил.

7.2 Съществуващ PDF

При отворен съществуващ PDF записът се делегира на QpdfBackend. Реалната имплементация добавя /FreeText annotation обекти с appearance stream. Това запазва оригиналните страници и добавя редакциите като PDF структура.
